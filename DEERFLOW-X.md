# DeerFlow-X：多项目协作与商业化工作流引擎

## 背景
DeerFlow 2.0 (bytedance) 是单会话研究助手，缺乏多项目、团队协作、商业变现能力。

## 目标
在 DeerFlow 基础上二次开发，构建 **AI 团队协作操作系统**，让多个 AI agent 像真实公司团队一样协作开发项目。

## 2026 下半年商业方向
**"Solo Prenuers OS"** —— 一个 AI agent 团队，替代 3-10 人初创团队
- 一个人 + AI 团队 = 独立开发者工作室
- 目标用户：独立开发者、Micro SaaS 创作者、副业创业者
- 变现路径：Pro 订阅 ($19/月) + Team 协作空间 ($49/月)

## 核心功能（二次开发）
1. **多项目管理系统** — 支持同时跑多个项目，每个项目独立记忆、代码库、目标
2. **团队角色系统** — CEO/架构师/程序员/测试/运维，角色分工、任务派发
3. **进度追踪仪表盘** — 实时看每个 agent 在干啥、做到哪一步
4. **自动化部署链** — 一键从需求到线上，支持 Vercel/Railway/Supabase
5. **知识库积累** — 项目决策、技术选型自动记录，新人加入秒上手
6. **商业化模块** — 内置订阅计费、用户管理、账单系统

## 技术栈
- 核心：Python + LangGraph（继承 DeerFlow）
- 前端：React + TailwindCSS (Shadcn)
- 数据库：Supabase（Auth + Postgres + Realtime）
- 部署：Railway / Vercel
- 支付：Stripe
