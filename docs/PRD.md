# SoloPreneurs OS — 产品需求文档 v1.0
## 愿景
让一个人 + AI团队 替代 3-10人 初创工作室。
## 用户故事
**F1: 任务下发** — 用户输入自然语言任务 → 系统分解 → 分配Agent → 实时进度
**F2: Agent团队管理** — 可配置Agent角色，独立开关，完整日志
**F3: 工作流模板** — 市场调研/内容创作/代码开发三大模板
**F4: 结果交付** — 汇总展示，一键导出，历史版本
## 验收标准
- [x] 3秒内开始执行
- [x] 5个Agent并行
- [x] 工作流可保存/复用
- [x] 结果可导出
## 核心接口
POST /api/tasks — 创建任务
GET /api/tasks/{id} — 查询状态
WebSocket /api/ws/tasks/{id} — 实时推送
